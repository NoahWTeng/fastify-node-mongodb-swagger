# Node - Fastify - Mongodb - Swagger

Building a Minimalist REST API Templeta with Node, Mongodb, Fastify and Swagger

## Installation
```
yarn 
yarn dev
```
## Create 50 fakes products
```
yarn seed
```
## Routes
```
# Swagger
http://localhost:3050/docs/ 
# Products
http://localhost:3050/api/v1/product
```
